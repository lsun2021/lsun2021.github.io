MacOS Hexo +Github 搭建自己的博客网站

1. 安装准备：

2.GitHub仓库创建

![image-20220324174616938](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324174616938.png)

新建仓库：New repository

创建仓库的名字必须为 用户名.github.io ,

![image-20220324174736136](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324174736136.png)

比如我的用户名是lsun2021，仓库名就必须是 lsun2021.github.io 。 这一点很关键，后续的 对外地址就是 www.lsun2021.github.io 。 否则会出现找不到地址的情况。

3.配置SSH Key

ssh-keygen -t rsa -C "your_email@example.com"

会在本地生成一个/Users/用户名/.ssh 文件夹，里面有 id_rsa 和 id_rsa.pub 里面有生成两个文件，打开 id_rsa.pub 文件，将里面的内容全选复制， 打开 Github -> 点击账号 ->Settings ->SSH and GPG keys ->New SSH Key

这样就可以添加了SSH Key

4. 搭建仓库

在本地新建一个文件 Hexo，打开控制台，

cd /Users/xxx/Developer/Hexo

输入： hexo init 初始化

![hexo01](/Users/wangxin/Library/Application Support/typora-user-images/hexo01.png)

这样就在你的目录Hexo 中初始化好了hexo 的代码，大家生成的都是一样的。

![image-20220324170728392](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324170728392.png)

打开 _config.yml 文件，滑动到底部，修改：

deploy: type: ''

修改后：

deploy: type: git repository: https://github.com/用户名/用户名.github.io.git branch: main

repository: GitHub仓库地址

branch: 新版的是main 分支，不是master

注意：GitHub仓库地址， 用户名前后一定要一致，要不然会出现找不到的问题，

eg: https://github.com/lsun2021/lsun2021.github.io.git

然后， 当前窗口输入命令：npm install hexo-deployer-git --save

![image-20220324171550180](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324171550180.png)

安装hero 对于git 的部署工具，

然后，hexo g

![image-20220324172441648](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324172441648.png)

hexo d

![image-20220324172745977](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324172745977.png)

如果出现以下无法访问的情况：

重复试试几次，还不行的话可能是本地的github.com的Ip不能访问，打开本地hosts文件，查询可用的IP。

在https://www.ipaddress.com/上分别搜索
github.com、github.global-ssl.fastly.net ，然后替换上去。

再试试：hexo d ,运行成功了

![image-20220324173626919](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324173626919.png)

出现 INFO Deploy done: git 表示成功，github刷新一下。

![image-20220324173756703](/Users/wangxin/Library/Application Support/typora-user-images/image-20220324173756703.png)

Finish
